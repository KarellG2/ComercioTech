# ComercioTech

## Contexto

ComercioTech es una empresa en proceso de expansión y necesita modernizar su infraestructura de base de datos para el manejo eficiente de la cantidad de clientes y transacciones, por lo que necesita una solución robusta, segura y escalable para la gestión de los datos de clientes, productos y pedidos.

Esta solución es una aplicación de escritorio desarollada para administrar clientes, productos y pedidos mediante la utilización del gestor de base de datos MongoDB. El sistema permite realizar operaciones CRUD, autenticación de usuarios y gestión centralizada de la información comercial.

## Objetivos

1. Escalabilidad: La solución ofrece a la organización ComercioTech la capacidad de escalar la cantidad de datos que esta almacena sin problemas.

2. CRUD: Creación, consulta, actualización y eliminación de registros de clientes, productos y pedidos

3. Intuitividad: Interfaz gráfica de usuario simple y fácil de comprender

4. Seguridad: Cifrado de datos en reposo y en tránsito, con login seguro 


## Tecnologias Utilizadas

2. PyQt6

3. MongoDB

4. PyMongo

-   Gestión de Clientes
